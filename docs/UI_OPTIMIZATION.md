# 练习设置界面优化说明

## 🎨 优化内容

### 1. AI 开关区域全面改版

#### 优化前的问题
- ❌ 文字和开关挤在一起，布局混乱
- ❌ 描述文字样式单调，不够醒目
- ❌ 缺少视觉层次感
- ❌ 与整体风格不协调

#### 优化后的效果
- ✅ **渐变背景卡片**：淡紫到淡粉的柔和渐变
- ✅ **清晰的层次结构**：标题、开关、描述分层展示
- ✅ **动画效果**：
  - 🤖 机器人图标轻微跳动（bounce 动画）
  - 背景光晕脉冲效果（glow 动画）
  - 点击反馈动画
- ✅ **视觉增强**：
  - 左侧边框强调色（紫色）
  - 柔和阴影提升层次
  - Emoji 图标增加趣味性

### 2. 整体设置面板优化

#### 标题区域
- 添加设置图标（⚙️）
- 底部分隔线
- 统一字体大小和粗细

#### 选择器（Picker）
- 柔和的背景色 (#f8f9fa)
- 细边框设计
- 下拉箭头指示符
- 点击反馈效果（active 态）

#### 滑块（Slider）
- 标准间距
- 数值实时显示

#### 按钮
- 保持原有的渐变紫色主题
- 圆角胶囊样式
- 一致的间距

## 🎯 样式细节

### AI 开关卡片
```css
背景：渐变（#f8f9ff → #fff5f8）
边框：2rpx 实线，#e0e7ff
圆角：16rpx
内边距：25rpx
阴影：rgba(102, 126, 234, 0.08)
```

### 动画效果

1. **机器人图标跳动**
   - 动画名：bounce
   - 时长：2秒
   - 效果：上下轻微跳动 6rpx

2. **背景光晕**
   - 动画名：glow
   - 时长：3秒
   - 效果：径向渐变脉冲

3. **选择器反馈**
   - 状态：active
   - 效果：背景变深，边框变紫

### 颜色方案
- 主题色：#667eea（紫色）
- 渐变色：#764ba2（深紫）
- 背景色：#f8f9ff, #fff5f8（淡紫、淡粉）
- 边框色：#e0e7ff（淡紫边框）
- 文字色：#333（主要），#666（次要）

## 📱 响应式适配

- 使用 rpx 单位自适应不同屏幕
- flex 布局确保元素对齐
- 相对定位实现图标叠加

## 🔧 技术实现

### 核心代码结构
```vue
<view class="form-item ai-switch-item">
  <!-- 头部：标题 + 开关 -->
  <view class="ai-switch-header">
    <view class="ai-switch-title">
      <text class="ai-icon">🤖</text>
      <text class="label ai-label">AI 智能出题</text>
    </view>
    <switch :checked="useAI" @change="onAISwitchChange" />
  </view>
  
  <!-- 描述框 -->
  <view class="ai-description-box">
    <text class="ai-description">
      ✨ 开启后使用 AI 生成...
    </text>
  </view>
</view>
```

### 关键样式
- `position: relative` + `overflow: hidden` 实现光晕效果
- `::before` 伪元素创建动画背景
- `z-index` 控制层级关系
- CSS 动画关键帧（@keyframes）

## 🎨 设计理念

1. **视觉层次**：通过背景、边框、阴影创建层次感
2. **微交互**：细微动画提升趣味性和生动感
3. **一致性**：所有元素遵循统一的设计语言
4. **可用性**：清晰的标识和描述，易于理解
5. **美观性**：柔和的配色和圆润的圆角
